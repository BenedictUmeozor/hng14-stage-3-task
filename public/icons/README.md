# PWA Icons

This directory should contain the PWA icons for the Habit Tracker application.

## Required Files

- `icon-192.png` - 192x192px PNG icon
- `icon-512.png` - 512x512px PNG icon

## How to Generate Icons

### Option 1: Use the HTML Generator (Recommended)

1. Open `scripts/generate-icons.html` in a web browser
2. Click "Download 192x192 Icon" and save as `icon-192.png` in this directory
3. Click "Download 512x512 Icon" and save as `icon-512.png` in this directory

### Option 2: Use Online Tools

- [Favicon Generator](https://www.favicon-generator.org/)
- [Real Favicon Generator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://www.pwabuilder.com/)

### Option 3: Create Manually

Use any image editor (Photoshop, GIMP, Figma, etc.) to create:
- A 192x192px PNG with the app icon
- A 512x512px PNG with the app icon

The current design uses:
- Background color: #3b82f6 (blue)
- Icon: White checkmark in a circle
- Rounded corners for a modern look

## Icon Design Guidelines

- Keep it simple and recognizable
- Use high contrast for visibility
- Ensure it looks good at small sizes
- Consider using the app's theme color (#3b82f6)
- Make it distinctive from other habit tracking apps
